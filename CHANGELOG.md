# Change Log

All notable changes to the "react-native-jsx-helper" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.0] - 2025-12-28

### Added
- Initial stable release.
- **Smart Detection**: Automatically detect JSX elements when the cursor is inside a tag.
- **Configurable Wrappers**: Fully customizable components, imports, and attributes.
- **Framework Presets**: Built-in defaults for React, React Native, and HTML.
- **Intelligent Unwrapping**: Safely remove wrappers while preserving or removing children.
- **Custom Element Wrapping**: Dynamic snippet support for wrapping with any element name.
- **Auto-Formatting**: Integrated with VS Code's formatting system.
